# node-REST-api
## a CRUD REST api created using vanilla node js

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install
yarn install

# Run in develpment
npm run dev
yarn run dev

# Run in production
npm start
yarn start
```
